Chemical Component Sketch Tool

The form below accepts input options to launch a 2D chemical sketch tool. This sketch tool can be used to build an annotated chemical component definition, or to build a query molecule for a substructure search of the chemical component dictionary. You may sketch a molecule from scratch or start a sketch using the chemical description in one of the following forms: an mmCIF format chemical component, a MOL/SDF file, PDB formatted coordinates, or a SMILES string.
