Help Page




"Load" button: Load a dictionary (or a saved) ligand.

"Browse" button: Browse your desktop's file system for a chemComp mmCif file to upload.

"Upload" button: Upload a chemComp mmCif file from your desktop's file system.

"Align" button: Realign the 2-D diagram to an optimal nice looking layout.

"Add Hydrogens" button: Will add hydrogens to the model.

"Reset windows" button: All sub-windows of this interface may be moved and resized. Pressing this button will redisplay all windows in their default location and size.

"New code" button: This will regenerate a new id code that is not currently in use.

"Save" button:This will save your current work in the server work-area. In following visits the use of the "Load" button will retrieve the saved work.`

"Clean up" button: Cleans up all saved ligands in the server work-area.

"Commit to CVS" button: Adds a new ligand to the repository.

More detailed documentation can be found here.

"Chemical diagram" panel

Allows one to display/edit the ligand in 2D using the buttons at the bottom of the panel.

"Edit" button: Indicates that clicking on an atom or bond will either add a partular atom or change the bond type.

"Select" button: Selects bonds/atoms so they can be moved in the 2D view.

"Delete" button: Clicking on this and then a particular atom/bond will delete it.

"Clear" button: Clears the work area.

"Bonds" pulldown: Indicates the type of bond to add.

"Element" pulldown: Indicates the atom type to add.

"Labels" toggle: Turns on/off the display of atom names.

"Hydrogens" toggle: Turns on/off the display of hydrogens

"Lock" toggle: Turns on/off the ability to grab and move atoms and bonds in the 2-D diagram (in order to avoid accidental changes).

"Undo/Redo" buttons: Undos (Redos) the last edit. Can be pressed repeatedly.

Molecule Panel

Displays CIF header for a ligand. May be edited.

"3-D model" panel

Displays the 3D representation of the model in JMOL. This is usually based on the model coordinates.

"3-D model" panel

Displays the ideal representation of the current model. Geometry is optimized compared to the "3-D view"

"Atoms" table

Displays the atoms present in the ligand. One may change the values by pull down on each element. Changes in ATOM_ID are reflected in the "Bonds" table.

"Bonds" table

Displays the connectivity of individual atoms. One may change the bond order of the entity.

"Audit" table

Displays the 'pdbx_chem_comp_audit' category. One may change the value(s) of the category.

"Synonym" table

Displays the 'pdbx_chem_comp_synonyms' category. One may change the value(s) of the category.

"Related" table

Displays the 'pdbx_chem_comp_related' category. One may change the value(s) of the category.

"Chemical format" panel

Displays the current entry.

"Format" pulldown: Selects the download and display format of the current entry.

"Download" button: Will download the current entry in the selected format.
